🍽️ Recipe Finder App

A ReactJS web application to search and explore meal recipes using TheMealDB API.
Users can search dishes, view recipe details, instructions, and watch tutorial videos. Fully responsive for desktop and mobile.

🔹 Features
* Search meals by name
* View recipe details with ingredients and instructions
* Watch related YouTube cooking videos
* Responsive design for all devices
* Clean, intuitive UI with images and info layout

⚡Technologies Used
* Frontend: ReactJS, CSS3, HTML5
* API: TheMealDB
* Other: Flexbox, Responsive design
